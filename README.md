# Indgram

DART(Data Analysis, Retrieval and Transfer System) 기업 정보 추출 및 분석 도구

## 설명

한국 기업들의 공시 정보를 DART API를 통해 수집하고 분석하는 프로젝트입니다.

## 설치

```bash
# uv를 사용하여 의존성 설치
uv sync
```

## 사용법

```bash
python main.py
```

## 의존성

- dart-fss: DART API 클라이언트
- pandas: 데이터 처리
- numpy: 수치 계산
- networkx: 네트워크 분석
- jupyter: 노트북 환경
