# 비상장주식 가치 평가 도구 (Sang-Jeung-Se Law Based)

이 프로그램은 상속세 및 증여세법(상증세법)에 기초하여 비상장주식의 1주당 평가액을 계산하는 CLI 도구입니다.

## 주요 기능
- **순손익가치(A) 계산**: 최근 3개년 순손익액의 가중평균 (3:2:1) x 10
- **순자산가치(B) 계산**: 자산총계 / 발행주식수
- **가중평균가치(C) 계산**: (순손익가치 * 3 + 순자산가치 * 2) / 5
- **최종 평가액(P) 결정**: max(가중평균가치, 순자산가치의 80%)

## 실행 방법 (uv 환경)
터미널에서 다음 명령어를 실행하세요:
```bash
uv run python request/valuation_cli.py
```

## 윈도우용 실행 파일(.exe) 만들기
이 프로그램을 윈도우에서 단독 실행되는 `.exe` 파일로 만드려면 `PyInstaller`를 사용할 수 있습니다.

1. **PyInstaller 설치** (윈도우 터미널):
   ```cmd
   pip install pyinstaller rich pandas
   ```

2. **빌드**:
   ```cmd
   pyinstaller --onefile request/valuation_cli.py
   ```
   *빌드가 완료되면 `dist` 폴더 안에 `valuation_cli.exe` 파일이 생성됩니다.*

## 데이터 입력 방식
1. **예시 데이터 실행**: 코드에 내장된 예시 값으로 즉시 결과를 확인합니다.
2. **직접 데이터 입력**: 연도별 발행주식수, 자기자본, 당기순이익을 직접 입력하여 계산합니다.
