# 포털 코드 해설 (Portal Code Reference)

---

## 한 줄 요약

각 코드는 **독립된 웹 포털(사이트)**을 의미합니다.
스프레드시트에서 해당 컬럼에 `1`이면 그 포털에 도구가 노출되고, `0`이면 숨겨집니다.

## 작동 원리

```
마스터 스프레드시트 (Tax_Hub 시트)

         KC  HUB  SPK  CEN  TA  TR  TP  TX  DD
         [5] [6]  [7]  [8]  [9] [10][11][12][13]  ← 컬럼 인덱스
         ─── ──── ──── ──── ─── ─── ─── ─── ───
AI 세법    1    1    1    1   1   1   1   1   1   ← 모든 포털에 노출
포렌식AI   1    1    1    0   0   1   1   1   1   ← CEN, TA에는 미노출
난방비     0    0    1    0   0   0   0   0   0   ← SPK(관리자)에만 노출
```

### 읽는 법
- 각 포털은 자기 컬럼 번호를 알고 있음 (예: CEN = 8번 컬럼)
- API로 스프레드시트를 읽은 후, **자기 컬럼이 `1`인 행만 필터링**
- 필터링된 도구만 화면에 카드로 렌더링

## 새 포털 추가 시 (예: 신문사용)

```
기존:  KC | HUB | SPK | CEN | TA | TR | TP | TX | DD
추가:  KC | HUB | SPK | CEN | TA | TR | TP | TX | DD | NEWS
                                                        ↑
                                                   컬럼 인덱스: 14
```

1. 스프레드시트에 `NEWS` 컬럼 추가
2. 신문사 정회원에게 보여줄 도구 행에 `1` 세팅
3. 새 Vercel 앱 배포 (columnIndex = 14로 필터링)
4. 완료 — 기존 시스템 변경 없음
